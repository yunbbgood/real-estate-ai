# 房仲 AI OS - Product Requirements Document
# 1. 專案目標

打造一套給台灣房仲使用的 AI 工作平台，協助處理客戶管理、案件管理、AI 估價、謄本 OCR、文宣生成與短影音腳本等工作。
## 2. 系統模組架構

房仲 AI OS V1.0 分成以下模組：

### 1. Dashboard 首頁
- 今日待辦
- 最近案件
- AI提醒
- 快速功能入口

### 2. 案件中心
- 建立案件
- 我的案件
- 案件詳情
- 編輯案件
- 案件狀態管理

### 3. OCR 中心
- 建物謄本 OCR
- 土地謄本 OCR
- 權狀 OCR
- 實價登錄資料解析

### 4. AI 估價中心
- OCR 資料整理
- 行情比對
- AI 建議售價
- 價格區間
- 估價報告

### 5. 文宣中心
- 房源文案
- 社群貼文
- LINE文案
- 海報文字
- 影片標題

### 6. 短影音中心
- 短影音腳本
- 分鏡腳本
- 即夢影片提示詞
- 封面文案

### 7. CRM 客戶中心
- 買方客戶
- 屋主客戶
- 回訪提醒
- 帶看紀錄
- 成交追蹤
